## How to Use This Repo

1. **Learning** - Go topic by topic in `01-DSA/`. Read `notes.md` first, then solve problems in `problems/`.
2. **Practice** - Add your solutions in the `problems/` folder of each topic.
3. **Revision** - Before interviews, use `Revision/cheatsheets/` for quick review.
4. **Track Progress** - Check off items in the [Roadmap](./01-DSA/Roadmap.md).
