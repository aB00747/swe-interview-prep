# Two Pointer & Sliding Window

## Two Pointer
- Opposite ends (sorted array problems)
- Same direction (fast-slow pointer)
- Use when: sorted array, find pair, remove duplicates

## Sliding Window
- **Fixed size**: window of size K, slide one step at a time
- **Variable size**: expand/shrink based on condition
- Use when: subarray/substring with some constraint

## When to Use
- "Find subarray with sum = K" -> Sliding window
- "Find pair with sum = target" -> Two pointer
- "Longest substring without repeating" -> Variable sliding window
- "Detect cycle in linked list" -> Fast-slow pointer
