# Stack & Queue

## Stack (LIFO)
- Push: O(1), Pop: O(1), Peek: O(1)
- Use cases: parentheses matching, undo, DFS, monotonic stack

## Queue (FIFO)
- Enqueue: O(1), Dequeue: O(1), Front: O(1)
- Use cases: BFS, task scheduling, sliding window maximum

## Variants
- Monotonic Stack (next greater/smaller element)
- Deque (double-ended queue)
- Priority Queue (see Heap)
- Circular Queue
