# Arrays

## Key Concepts
- Contiguous memory allocation
- Fixed size (static) vs dynamic arrays
- Index-based access: O(1)

## Operations & Complexity
| Operation | Time Complexity |
|-----------|----------------|
| Access | O(1) |
| Search | O(n) |
| Insert (end) | O(1) amortized |
| Insert (middle) | O(n) |
| Delete | O(n) |

## Common Patterns
- Two pointer
- Prefix sum
- Sliding window
- Kadane's algorithm (max subarray)
- Dutch National Flag (sort 0,1,2)
