# Heap / Priority Queue

## Key Concepts
- Complete binary tree stored as array
- Min-Heap: parent <= children
- Max-Heap: parent >= children

## Operations
| Operation | Time |
|-----------|------|
| Insert | O(log n) |
| Extract min/max | O(log n) |
| Peek min/max | O(1) |
| Build heap | O(n) |

## Common Patterns
- Top K elements (min-heap of size K)
- Merge K sorted lists
- Median of stream (two heaps)
- K closest points
