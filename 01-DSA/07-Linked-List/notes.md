# Linked List

## Types
- Singly Linked List
- Doubly Linked List
- Circular Linked List

## Operations & Complexity
| Operation | Time |
|-----------|------|
| Access by index | O(n) |
| Insert at head | O(1) |
| Insert at tail | O(n) / O(1) with tail pointer |
| Delete | O(n) |
| Search | O(n) |

## Common Patterns
- Fast-slow pointer (find middle, detect cycle)
- Reverse linked list (iterative & recursive)
- Merge two sorted lists
- Remove nth node from end
